Backend + Frontend start (Docker, PowerShell)

1) Go to repo
cd F:\DE_intern\probablyfresh-analytics-platform

2) Build images (after any backend/frontend dependency changes)
docker compose --env-file .env build backend frontend

3) Start backend + frontend
docker compose --env-file .env up -d backend frontend

4) Check containers
docker compose --env-file .env ps backend frontend

5) Open apps
- Frontend UI: http://localhost:5173
- Backend API: http://localhost:8001/api
- API docs: http://localhost:8001/api/docs/

6) Useful logs
docker compose --env-file .env logs -f --tail=200 backend
docker compose --env-file .env logs -f --tail=200 frontend
